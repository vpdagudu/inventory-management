Factory Inventory Management System

A full-stack web application for managing factory inventory, orders, demand forecasting, spending analytics, and reporting across multiple warehouses.

Tech Stack

Frontend

  • Vue 3 (Composition API) 3.4.21
  • Vite 5.2.0
  • Vue Router 4.3.0
  • Axios 1.6.7
  • i18n EN / JP

Backend

  • Python 3.11+
  • FastAPI 0.110.0+
  • Uvicorn (ASGI) 0.24.0+
  • Pydantic 2.5.0+

Data Layer

  • JSON files 7 files
  • In-memory storage at startup
  • No external DB read-only

Testing & Tooling

  • pytest 8.0.0+
  • httpx (test client) 0.27.0+
  • pytest-cov 4.1.0+
  • 55 passing tests 100%

Frontend Views

Dashboard

KPI metrics, order health donut chart, revenue tracking against $800K/month goal

Inventory

Stock levels across San Francisco, Tokyo, London warehouses with search and low-stock alerts

Orders

Order tracking with status pipeline: Delivered, Shipped, Processing, Backordered

Demand Forecasting

Current vs. forecasted demand with trend indicators (Increasing, Stable, Decreasing)

Spending

Financial analytics by category: Procurement, Operational, Labor, Overhead

Reports

Quarterly performance summaries and month-over-month trend analysis

Backlog

Unfulfilled orders with priority levels and linked purchase order status
